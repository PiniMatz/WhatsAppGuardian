# WhatsApp Guardian - שומר הוואטסאפ

לוח בקרה להורים ואפליקציית אנדרואיד (סלולרית) לסינון וזיהוי שיחות אלימות, בריונות ברשת, חרמות או פגיעה עצמית בוואטסאפ בעברית.

הפרויקט מורכב משני חלקים:
1. **שרת מקומי / ענן (Backend)**: שרת Node.js שמקבל התראות, מנתח אותן באמצעות ה-API של Gemini ומנהל את רשימת מילות המפתח המשותפת דרך Firebase.
2. **אפליקציית קצה לילד (Android Client)**: שירות נגישות (Accessibility Service) המאזין לתוכן המוצג על המסך ומסנן מילים חשודות מקומית לפני שליחתן לניתוח בשרת.

---

## מבנה הפרויקט
* `/server`: קוד שרת ה-Express וה-API של Gemini וכן דף לוח הבקרה של ההורים.
* `/client`: קוד האפליקציה המותקנת בטלפון של הילד (Kotlin Android).

---

## הוראות הרצה - שרת (Backend)

1. היכנס לתיקיית השרת:
   ```bash
   cd server
   ```
2. התקן את התלויות:
   ```bash
   npm install
   ```
3. הגדר משתני סביבה בקובץ `.env`:
   צור קובץ בשם `.env` בתיקיית `server` והכנס אליו את המפתח של Gemini:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   PORT=3000
   ```
4. **הגדרת Firebase** (אופציונלי אך מומלץ):
   * היכנס לקונסול של Firebase ופתח פרויקט חדש.
   * צור מסד נתונים מסוג Firestore.
   * גש להגדרות הפרויקט -> Service Accounts (חשבונות שירות) ויצר מפתח פרטי חדש בפורמט JSON.
   * שמור את הקובץ שהורד בתיקיית `server` תחת השם `firebase-service-account.json`.
   * *הערה*: אם הקובץ לא יסופק, השרת יפעל במצב של זיכרון זמני (Mock Database) לצורך הדגמה ובדיקה בלבד.
5. הרץ את השרת:
   ```bash
   npm start
   ```
   לוח הבקרה יהיה זמין בכתובת: `http://localhost:3000`

---

## הוראות הרצה - אפליקציית הילד (Android)

1. פתח את תיקיית `/client` בתוך Android Studio.
2. הגדר את כתובת ה-IP של השרת שלך בקובץ `AlertSender.kt` (ממוקם ב-`client/app/src/main/java/com/guardian/client/utils/AlertSender.kt`):
   * אם אתה מריץ על אמולטור, הכתובת `http://10.0.2.2:3000` תעבוד ישירות.
   * אם אתה מריץ על מכשיר אמיתי, שנה את הכתובת ל-IP של המחשב שלך ברשת המקומית (לדוגמה `http://192.168.1.50:3000`).
3. בנה והתקן את ה-APK על הטלפון של הילד.
4. **הפעלת הרשאות בטלפון הילד**:
   * פתח את האפליקציה בטלפון והזן את שם הילד (נועם, שירה וכו') ולחץ שמור.
   * לחץ על כפתור הנגישות (Accessibility) והפעל את שירות **WhatsApp Guardian**.
     *(באנדרואיד מודרני, ייתכן שתצטרך להיכנס להגדרות היישום בטלפון, ללחוץ על 3 נקודות בפינה העליונה ולבחור "אפשר הגדרות מוגבלות / Allow Restricted Settings" כדי לאפשר את הנגישות).*
   * לחץ על כפתור גישה להתראות והפעל גישה עבור **WhatsApp Guardian**.
